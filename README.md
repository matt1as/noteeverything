# NoteEverything

**NoteEverything** is a beautiful, modern, GitHub-backed note-taking application designed for developers. It combines the speed of local-first editing with the reliability of Git version control.

## Getting Started

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Configure GitHub OAuth** - See [configuration guide](docs/getting-started/configuration.md)

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Open** [http://localhost:3000](http://localhost:3000)

For detailed instructions, see the [Installation Guide](docs/getting-started/installation.md).
