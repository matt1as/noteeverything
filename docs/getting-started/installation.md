# Installation

This guide will help you install and set up NoteEverything on your local machine.

## Prerequisites

- Node.js 18 or higher
- A GitHub account
- npm or yarn package manager

## Steps

### 1. Clone the Repository

```bash
git clone https://github.com/yourusername/note-everything.git
cd note-everything
```

### 2. Install Dependencies

```bash
npm install
```

That's it! You're ready to move on to [configuration](./configuration.md).
