# Quick Start

Get up and running with NoteEverything in minutes.

## Starting the Development Server

After completing [installation](./installation.md) and [configuration](./configuration.md), run:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## First Steps

### 1. Sign In

Click the **Sign in** button in the sidebar to authenticate with GitHub.

### 2. Configure Your Repository

1. Click the **Settings (Gear)** icon in the sidebar
2. Enter your GitHub username
3. Enter the name of the repository you want to sync with (e.g., `my-notes`)
4. The repository must already exist on GitHub

### 3. Create Your First Note

1. Click the **+** button to create a new note
2. Start typing in the editor
3. Your note is automatically saved to local storage

### 4. Sync to GitHub

Click the **Push (Cloud Arrow)** button in the sidebar footer to commit your changes to GitHub.
