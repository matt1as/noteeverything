# User Guide

Learn how to make the most of NoteEverything.

## Authentication

### Signing In

1. Click the **Sign in** button in the sidebar
2. You'll be redirected to GitHub for authorization
3. Grant permissions to the application
4. You'll be redirected back to NoteEverything

### Signing Out

Click your profile picture in the sidebar and select **Sign out**.

## Managing Notes

### Creating Notes

1. Click the **+ (Plus)** button in the sidebar
2. A new untitled note will be created
3. Click the note title to rename it
4. Start typing in the editor

### Editing Notes

- Click any note in the sidebar to open it
- Changes are saved automatically to LocalStorage
- Use the formatting toolbar for rich text options

### Organizing Notes

- Drag and drop notes to reorder them
- Create nested structures by indenting notes
- Use meaningful names for easy navigation

## GitHub Synchronization

### Initial Setup

1. Create a repository on GitHub (e.g., `my-notes`)
2. Click the **Settings (Gear)** icon in NoteEverything
3. Enter your GitHub username
4. Enter the repository name
5. Click **Save**

### Pushing Changes

1. Click the **Cloud Arrow (Push)** button in the sidebar footer
2. Your notes will be committed to GitHub as Markdown files
3. The folder structure in your repo will match your note hierarchy

### Pulling Changes

If you edit notes directly on GitHub or from another device:

1. Click the **Pull** button (if available in settings)
2. Or refresh the page to fetch latest changes
3. Local changes will be merged with remote changes

## Keyboard Shortcuts

- **Ctrl/Cmd + B:** Bold text
- **Ctrl/Cmd + I:** Italic text
- **Ctrl/Cmd + Shift + C:** Code block
- **Ctrl/Cmd + S:** Sync to GitHub (if configured)

## Tips and Best Practices

1. **Regular Syncing:** Push your notes to GitHub frequently for backup
2. **Meaningful Names:** Use descriptive note titles for better organization
3. **Hierarchical Structure:** Group related notes in folders/categories
4. **Markdown Benefits:** Since notes are Markdown, you can edit them anywhere
5. **Version Control:** Check your GitHub repository for full note history
