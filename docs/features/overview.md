# Features Overview

NoteEverything combines modern note-taking with powerful Git-based version control.

## Core Features

### Local-First Architecture

- **Instant Saving:** Notes are saved to browser LocalStorage immediately as you type
- **Zero Latency:** No waiting for server responses
- **Offline Support:** Continue working even without an internet connection

### GitHub Sync

- **Version Control:** Full Git history for all your notes
- **Backup:** Automatic backup to your private GitHub repository
- **Markdown Storage:** Notes stored as plain Markdown files
- **No Vendor Lock-in:** Your data remains portable and accessible

### WYSIWYG Editor

Powered by TipTap, the editor supports:

- Rich text formatting (bold, italic, code)
- Lists (ordered and unordered)
- Code blocks with syntax highlighting
- Headings and paragraphs
- Real-time preview

### Modern User Interface

- **Linear-Inspired Design:** Clean, minimal aesthetic
- **Zinc Theme:** Professional color scheme
- **Dark/Light Mode:** Automatic theme switching
- **Responsive Layout:** Works on desktop and mobile

### Hierarchical Organization

- **Recursive Tree Structure:** Unlimited nesting of notes
- **Folder-Like Organization:** Group related notes together
- **Easy Navigation:** Collapsible sidebar with tree view

## Technical Highlights

- **TypeScript:** Full type safety
- **App Router:** Modern Next.js 14 architecture
- **NextAuth:** Secure GitHub authentication
- **Tailwind CSS v4:** Utility-first styling with Typography plugin
